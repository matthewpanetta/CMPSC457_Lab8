CMPSC 457 Lab #8

By: Matthew Panetta

Instructions

In this lab I needed to:
  • Create the letter 'R' in an 8x16 bit pattern
  • Use a 2D array to represent a bitmap image

My Implementation

  • I made a 2D array of size 16. Since the bit pattern is 8x16, each array element (byte) represents one row.
  • The bytes form the letter using hexadecimal digits.
  • The letter is drawn in the center of the screen.
